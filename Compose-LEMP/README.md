#Docker-Compose LEMP ENV

Installation d'un environnement LEMP
* Linux (Debian Jessie)
* Engine-X (Nginx)
* MariaDB
* PHP 7.0
* PHPMyAdmin  

PHPMyAdmin est accessible sur le port [8088](http://0.0.0.1:8088)
Nginx est accessible sur le port [8082](http://0.0.0.1:8082)

```bash
docker-compose up -d --build
```
